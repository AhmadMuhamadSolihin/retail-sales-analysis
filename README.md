# Retail Sales Analysis

## Project Overview
This project analyzes retail transaction data to identify sales patterns, customer behavior, and business insights.

## Tools
Python
Pandas
Matplotlib
PowerBI

## Dataset
Supermarket sales dataset containing 1000 transactions.

## Business Questions
- Which product generates the highest sales?
- Which branch performs the best?
- What time has the highest transactions?
- What customer segment generates the most revenue?

## Key Insights

1. Electronic Accessories is the best selling product category.
2. Food and Beverages generate the highest profit.
3. Branch C has the highest total sales.
4. Peak transactions occur at 19:00.
5. E-wallet is the most used payment method.

## Dashboard
PowerBI dashboard visualization.

## Business Recommendations
- Increase staff during peak hours
- Focus marketing on best selling products
- Improve service quality in lower rated branches
